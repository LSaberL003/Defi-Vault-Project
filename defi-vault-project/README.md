Project: DeFi Vault

Features:
- ERC20 token used as collateral
- Users can approve and deposit ERC20 tokens into the vault
- Users receive vault shares after deposit
- Users can withdraw by burning vault shares
- A 1% withdrawal fee is sent to the admin
- Users receive a governance membership token after deposit
- Governance membership is revoked when all vault shares are withdrawn
- Frontend supports connect wallet, approve, deposit, withdraw, and balance display
